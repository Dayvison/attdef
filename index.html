What's new?

- Removed anti-macros system and all of its components.

- Removed old AC system and all of its components.

- The mighty new Anti-Cheat is now fully compatible.

- Added /p and /u commands for quick round pause/unpause.

- Added a new command /reloaddb to reload the SQLite database.

- Feature: Player replacement now is made into user-friendly dialogs.

- Feature: You should not get hit while picking weapons from gunmenu now.

- Feature: A sound is now played when a player makes a pause or an unpause request.

- Bug-fix: you're now given a parachute on round-unpause if you get one before pause/crash.

- Bug-fix: players now are re-spawned in their vehicles after crash or sudden leave.

- Fixed /afk bug allowing non-admins to set anyone to afk mode.

- Solved a weird and old issue regarding SQLite database loading.
